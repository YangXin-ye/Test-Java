# 一、前端
## 1、修改 Header.vue 文件进行给个人信息按钮添加点击事件
```vue
<el-dropdown-item @click.native="handleClick"></el-dropdown-item>
```
注销按钮添加点击事件并且绑定 handleClick 方法
```js
handleClick() {
      this.$message({
        message: '退出登陆成功',
        type: 'success'
      })
      localStorage.removeItem('token')
      // 跳转到首页
      this.$router.push('/login')
    },
```
methods 方法添加退出登陆函数
